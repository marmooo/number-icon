# Number Icon

[An icon drawing game to connect numbers](https://marmooo.github.io/number-icon/).

## Installation

1. `gitn clone vendor/ repos.lst` (see [gitn](https://github.com/marmooo/gitn))
2. `bash build-svg.sh`
3. `deno run --allow-read --allow-write listup.js`

## Build

```
bash build.sh
```
